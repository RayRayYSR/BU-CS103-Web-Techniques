Introduction & Explanation of the web page

This web page contains multiple modern front end developing techniques

1. Bootstrap settings: This page uses bootstrap settings. As a real-life front end web developer, efficiency and quality are crucial factors. Applying Bootstrap settings can provide a nice and neat responsive web layout. In addition, the coding speed will improve drastically since pre-set classes and elements are available. In addition, custom classes can be nested inside bootstrap class. In this case, the web layout and code style can be more personalized and unique. Please enjoy this responsive web page.

2. SaSS: Instead of using "dumb" CSS style coding. SaSS is used. CSS codes can become very long and have lots of nested layers. More importantly, CSS cannot have logics, variables and many other programming language features. SaSS solves all these issues. If you want to learn more about this SaSS structure in my layout, please make sure download all 4 files (index.html, lab5_assignment.css, lab5_assignment.css.map, lab5_assignment.scss). Do NOT just download .html file and .css file.

3. Web Animation: This web page doesn't introduce some complex packages for animations.(i.e. d3.js, Canvas). CSS and Computer Graphics theories are used. For example, hover your mouse on the button below. This button is built by CSS, keyframes and Beizer Curve Theory are applied to run the movement.
